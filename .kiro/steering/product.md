# Product Steering

## 目的
- Codex エージェント向けスキルを集約・運用するための中核リポジトリ
- スキルの発見性と運用ルールの一貫性を高め、迷いなく利用できる状態を作る
- 複数エージェント環境への配布・同期を安定化する

## 価値
- README を入口、各 `SKILL.md` を正として情報の重複と混乱を減らす
- 自作スキルと外部スキルを区別し、内部を優先して運用する

## コア機能
- スキル定義の管理（`SKILL.md` を中心としたルールと更新）
- スキルの同期・バンドル化（Nix / Home Manager を利用）
- ドキュメント品質の担保（mise タスクと CI による lint/format/validate）

## しないこと
- スキルの詳細設計をここで網羅しない（各 `SKILL.md` に委譲）
- 依存関係やファイル一覧の完全な列挙はしない（パターンのみ記載）
